E-WASTE LOCATOR

E-Waste Locator is a web application that helps users find nearby e-waste recycling facilities and manage electronic waste responsibly.

---

## FEATURES

* Locate nearby e-waste recycling centers
* View recycling facility details
* Simple and user-friendly interface

## INSTALLATION AND SETUP

1. Clone the repository
   git clone [https://github.com/tamil123tami/e-waste_locator.git](https://github.com/tamil123tami/e-waste_locator.git)

2. Navigate to the backend folder
   cd e-waste_locator/backend

3. Install dependencies
   npm install

4. Start the backend server
   node server.js

5. Open the frontend
   Open frontend/index.html in your web browser

## USAGE
