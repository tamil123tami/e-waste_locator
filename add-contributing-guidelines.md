## WELCOME

Thank you for taking interest in contributing to the E-Waste Locator project.
All contributions are welcome, including code, documentation, and suggestions.

---

## HOW TO CONTRIBUTE

1. Fork the repository to your GitHub account
2. Clone your fork to your local machine
3. Create a new branch for your changes
4. Make your changes and test them
5. Commit your changes with a clear message
6. Push the changes to your fork
7. Open a Pull Request to the main repository

---

## BRANCHING RULES

* Do not work directly on the main branch
* Create a new branch for each feature or fix
* Use meaningful branch names
  Example: fix-readme, add-validation, ui-improvement

---

## COMMIT MESSAGE GUIDELINES

* Use short and clear messages
* Describe what you changed

Examples:

* "Fix typo in README"
* "Add input validation to frontend"
* "Improve UI styling"

---

## CODE STYLE GUIDELINES

* Keep code clean and readable
* Use proper indentation
* Add comments where necessary
* Follow existing project structure

---

## ISSUES AND FEATURE REQUESTS

* Check existing issues before creating a new one
* Clearly describe the problem or feature request
* Add steps to reproduce bugs if possible

---

## PULL REQUEST RULES

* Explain what your changes do
* Link the related issue if available
* Be respectful during reviews
* Make requested changes if asked
